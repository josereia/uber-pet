# Uber PET

### How to build
* Manually
  ~~~bash
  mkdir build
  cd build
  cmake ..
  make -j
  ~~~
* Script(Linux and MacOS)
  ~~~bash
  chmod +x ./scripts/build.sh
  ./scripts/build.sh
  ~~~

### How to run
* Manually
  ~~~bash
  ./build/uber-pet
  ~~~
* Script(Linux and MacOS)
  ~~~bash
  chmod +x ./scripts/run.sh
  ./scripts/run.sh
  ~~~
